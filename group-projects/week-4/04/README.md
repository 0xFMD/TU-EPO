# Week 4 - Hash Example - PDF file with bitmap representation.

## Files

| File             | Role                                                                    |
| ---------------- | ----------------------------------------------------------------------- |
| [main.c](main.c) | Main program that generates a key and check if the PDF file is modified |

## Input

```bash
gcc main.c -o main
./main
```

## Output

When you run the program, it will read a PDF file, generate a key, and then represent the key using bitmap on the terminal to check if the PDF file modified or not.
