# Week 4 — Simple bitmap example in C

## Files

| File             | Role                                     |
| ---------------- | ---------------------------------------- |
| [main.c](main.c) | Main program that manipulates the bitmap |

## Input

```bash
gcc main.c -o main
./main
```

## Output

When you run the program, it will display the bitmap on the terminal. The output will show the visual representation of the bitmap with different colors and symbols.
