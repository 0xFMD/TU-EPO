# Week 4 — Simple generic doubly linked list implementation in C

## Files

| File             | Role                                                                |
| ---------------- | ------------------------------------------------------------------- |
| [main.c](main.c) | Main program that defines the generic doubly linked list operations |

## Input

```bash
gcc main.c -o main
./main
```

## Output

When you run the program, it will perform operations on a generic doubly linked list. The output will show the results of these operations, such as adding, and deleting nodes with different data types (e.g., integers, characters), as well as displaying the list.
