# Week 4 — Directed Graphs: Social Network

## Files

| File             | Role                                                           |
| ---------------- | -------------------------------------------------------------- |
| [main.c](main.c) | Main program that defines Directed Graphs for a Social Network |

## Compilation

```bash
gcc main.c -o main
./main
```

## Output

The program will create a directed graph representing a social network of users. Each user will have a list of followers and following nodes. The program will print out the username and the number of users they are following and who is following them.
