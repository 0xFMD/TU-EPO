# Week 4 — Doubly linked list / CRUD operations for Student records

## Files

| File             | Role                                                                            |
| ---------------- | ------------------------------------------------------------------------------- |
| [main.c](main.c) | Main program that defines the doubly linked list operations on a single pointer |
| [lst.c](lst.c)   | Same implementation but using double pointers                                   |

## Input

```bash
gcc main.c -o main
./main
```

## Output

When you run the program, it will perform CRUD operations on a doubly linked list of Student records. The output will show the results of these operations, such as adding, updating, and deleting student records, as well as searching and displaying the list of students.
